## 重要说明：免费疫苗 vs 自费疫苗

### 一类疫苗（免费必打）- 19种
这些是国家免疫规划疫苗，**必须接种**，完全免费：

1. **卡介苗（BCG）** - 出生时
2. **乙肝疫苗** - 3剂（出生、1月、6月）
3. **脊灰疫苗** - 3剂+加强（2月、3月、4月、4岁）
4. **百白破疫苗** - 3剂+加强（3月、4月、5月、18月）
5. **A群流脑疫苗** - 2剂（6月、9月）
6. **麻腮风疫苗** - 2剂（8月、18月）
7. **乙脑减毒活疫苗** - 1剂+加强（8月、2岁）
8. **甲肝减毒活疫苗** - 1剂（18月）
9. **A+C群流脑疫苗** - 1剂+加强（3岁、6岁）
10. **白破疫苗** - 加强（6岁）

### 二类疫苗（自费推荐）- 18种
这些是自费疫苗，**可选择接种**，但强烈推荐：

#### 最推荐（★★★★★）
- **13价肺炎球菌疫苗** - 3剂+加强（2月、4月、6月、12-15月）
  - 预防肺炎、脑膜炎、中耳炎
  - **不替代任何免费疫苗，是额外保护**

#### 强烈推荐（★★★★）
- **手足口病疫苗（EV71）** - 2剂（6月、7月）
  - 预防EV71病毒引起的手足口病重症
  
- **水痘疫苗** - 2剂（12月、4岁）
  - 预防水痘
  
- **流感疫苗** - 每年1剂（6月龄起）
  - 预防流行性感冒，每年接种

#### 推荐（★★★）
- **轮状病毒疫苗** - 口服，每年1剂（2月-3岁）
  - 预防轮状病毒腹泻

#### 可替代免费疫苗的自费疫苗
- **五联疫苗** - 3剂+加强（2月、3月、4月、18月）
  - **可替代百白破+脊灰疫苗**
  - 一针预防五种疾病，减少接种次数
  - 如果选择五联疫苗，就不需要打免费的百白破和脊灰了

- **Hib疫苗** - 3剂+加强（2月、4月、6月、18月）
  - 预防b型流感嗜血杆菌
  - **如果选择五联疫苗，就包含了Hib，无需单独接种**

## 关键区别

### 免费疫苗（一类）
- ✅ 国家免疫规划，必须接种
- ✅ 完全免费
- ✅ 在社区卫生服务中心接种
- ✅ 入托、入学时需要查验接种证

### 自费疫苗（二类）
- ⭐ 自愿选择，但强烈推荐
- 💰 需要自费
- 🔄 部分可以替代免费疫苗（如五联疫苗）
- 🛡️ 部分是额外保护（如13价肺炎、手足口）

## 接种时间表示例

### 2月龄宝宝可以接种：
**免费：**
- 脊灰疫苗第1剂

**自费（可选）：**
- 13价肺炎疫苗第1剂（额外保护）
- 五联疫苗第1剂（替代脊灰+百白破）
- Hib疫苗第1剂（如果没选五联）
- 轮状病毒疫苗

### 如果选择五联疫苗：
- 2月龄：五联疫苗第1剂（替代脊灰第1剂）
- 3月龄：五联疫苗第2剂（替代脊灰第2剂+百白破第1剂）
- 4月龄：五联疫苗第3剂（替代脊灰第3剂+百白破第2剂）
- 5月龄：**不需要打百白破第3剂**（已被五联替代）
- 18月龄：五联疫苗加强（替代百白破第4剂）

## 数据统计

```sql
-- 统计疫苗数量
SELECT
  category,
  CASE category
    WHEN 'free' THEN '一类疫苗（免费）'
    WHEN 'optional' THEN '二类疫苗（自费）'
  END as category_name,
  COUNT(*) as count
FROM vaccine_schedules
GROUP BY category;

-- 结果：
-- free (一类疫苗)：19种
-- optional (二类疫苗)：18种
```

## 常见问题

### Q: 自费疫苗必须打吗？
A: 不是必须的，但强烈推荐。特别是13价肺炎疫苗和手足口病疫苗。

### Q: 五联疫苗和免费疫苗冲突吗？
A: 不冲突，五联疫苗是替代方案。选择五联后，就不需要打免费的百白破和脊灰了。

### Q: 13价肺炎疫苗和免费疫苗冲突吗？
A: 不冲突，13价肺炎疫苗是额外保护，不替代任何免费疫苗。

### Q: 为什么有些疫苗在同一个月龄？
A: 不同疫苗可以同时接种（不同部位），例如2月龄可以同时接种脊灰疫苗（免费）和13价肺炎疫苗（自费）。

### Q: 如何选择？
A: 
1. **免费疫苗必须全部接种**
2. **自费疫苗根据经济条件选择**：
   - 预算充足：13价肺炎 + 手足口 + 水痘 + 流感
   - 预算有限：优先13价肺炎和手足口
   - 想减少接种次数：选择五联疫苗
