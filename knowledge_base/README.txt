# 母婴知识库索引

本知识库包含全面的母婴护理和育儿知识，适用于0-3岁婴幼儿的养育。所有内容基于世界卫生组织（WHO）、美国儿科学会（AAP）、中国营养学会等权威机构的指南和建议。

## 知识库文件列表

### 孕期相关

#### 12_产检时间表与注意事项.txt
**内容概要**：
- 完整的孕期产检时间表（孕早期、中期、晚期）
- 重要检查详解（NT、唐筛、大排畸、糖耐、胎心监护）
- 产检注意事项和准备工作
- 产检费用参考和医保报销

**适用场景**：
- 孕期产检规划
- 了解各项检查的意义
- 产检前准备
- 高危妊娠产检指导

#### 13_月子护理与产后恢复指南.txt
**内容概要**：
- 月子期注意事项（饮食、休息、卫生、环境）
- 产后身体恢复（子宫、恶露、伤口、体重、盆底肌）
- 产后心理健康（产后抑郁预防）
- 产后42天复查
- 月子误区辟谣

**适用场景**：
- 产后月子护理
- 产后恢复指导
- 预防产后抑郁
- 科学坐月子

- **孕期相关**：2个文件（产检、月子）

2. **补贴计算**：根据出生日期自动计算可领补贴
3. **办事指南**：证件办理流程和材料清单
4. **健康护理**：疾病护理、安全防护、营养喂养
5. **成长发育**：发育标准、早期教育、智力开发

## 使用建议

### 按需查询
根据宝宝当前月龄和遇到的具体问题，查询相应章节。

### 系统学习
建议新手父母系统阅读所有文件，建立完整的育儿知识体系。

### 定期复习
随着宝宝成长，定期复习下一阶段的内容，提前做好准备。

### 结合实际
知识库提供的是一般性指导，具体情况需结合宝宝个体差异和医生建议。

## 重要提醒

1. **个体差异**：每个宝宝发展速度不同，不要过度焦虑
2. **及时就医**：出现异常情况及时咨询医生
3. **科学育儿**：遵循科学指导，避免民间偏方
4. **安全第一**：始终把宝宝安全放在首位
5. **快乐成长**：让宝宝在爱和关怀中健康快乐成长
